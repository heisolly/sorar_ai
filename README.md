# Sorar AI üé≠

Sorar AI is a premium social skills training application designed to help users master difficult conversations through AI-powered roleplay simulations. Whether you're preparing for a business negotiation, improving your dating skills, or resolving family conflicts, Sorar AI provides a realistic, judgment-free environment to practice and grow.

## üöÄ Key Features

- **AI Roleplay Simulations**: Practice with diverse AI personalities across categories like Rizz (Dating), Business, Family, and Conflict Resolution.
- **Adaptive AI Modes**: Switch between Practice (you play yourself), Assistant (AI plays you to show you how), and Hybrid modes.
- **Real-time Coaching**: Get AI-powered suggestions during conversations to help you find the right words.
- **Empathy Avatar**: A reactive AI companion that provides emotional feedback and visual cues.
- **Personalized Dashboard**: Track your "Social Power Level", maintenance streaks, and daily insights.
- **Progress Tracking**: Detailed analytics to measure your communication improvement over time.

## üèÅ Getting Started

1. **Clone the repository**:

   ```bash
   git clone https://github.com/heisolly/sorar_ai.git
   ```

2. **Install dependencies**:

   ```bash
   flutter pub get
   ```

3. **Setup environment**:
   Create a `.env` file in the root directory with your Supabase credentials:

   ```
   SUPABASE_URL=your_project_url
   SUPABASE_ANON_KEY=your_anon_key
   ```

4. **Run the app**:
   ```bash
   flutter run
   ```
